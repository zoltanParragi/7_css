z-index

An elemet with higher z-index covers the elements with lower z-index or with no z-index.

z-index works only with the elements that have a position property (with any value).

overflow

shorthand property

  • visible (default)
  • hidden
  • auto (scrollbar where necessary)
  • scroll (x, y scrollbars)

flexbox

flex-wrap: wrap - elements with fix width will be placed in rows if one row is not enough. no-wrap is default

flex: 1; on flex child elements means that they will take equal width in their parent.

flex-direction: row, column, row-reverse, column-reverse

flex-flow: shorthand for flex-direction and flex-wrap

justify-content:(horizontal) flex-start, flex-end, center, space-around, space-between, space-evenly

align-items: (vertical) stretch, flex-start, flex-end, center

align-content: if elements are broken into at least 2 rows, vertical positions
