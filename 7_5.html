It is also usable for padding, margin, width, height, max-width,… !Remember: when em units are used on font-size, the size is relative to the font-size of the parent. When used on other properties, it’s relative to the font-size of the element itself. Problem: compoundin effect:

position

static(default), relative, absolute, fixed

  • static: it takes its natural place
  • relative: you can define its place (top, left, ...) relative to its natural static position
  • absolute: you can define its position (top, left, ...) relaive to its first positioned parent
  • fixed: relative to the window

If you want to use absolute you have to add relative to its parent or it will position itself to the root element(body).

z-index can be used only with elements where position is set.

If you move an item upward (with negative value eg.: top: -40px) you have to add margin to to element above to keep the desired space between them.
