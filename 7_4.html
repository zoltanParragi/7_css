Border, outline, text formatting

border

Shorthand property: with, type, color. The sequence is indifferent. One compulsory element is type, the other two is optional.

margin

can be used only on block elements.

auto can be used only if width is defined.

shorthand property ...(eg. margin { top leftRight bottom })

Absolute values: px, pt, cm, in ... Relative values: %, vw, vh

If property value is given in %, it refers to the WIDTH of the parent element.

vertical margins overlap:

horitontal margins does not overlap

padding

similar to margin

dimensions

max-width, max-height, min-width, min-height: If the width or height is defined in %, max-width, max-height, min-width, min-height can define a maximum or minimum value for eg. a picture to avoide too small or too large size (ugly pic)

box-model

... border-box

outline

It is a shorthand property (width, style, color). It is out of the border.

Any element can have an outline. Difference between outline and border: outline does not takes any place, it "floats" over anything. If you don't want outline to overlap any elements give larger margin to the element than outline-offset value.

outline-offset: CSS property sets the amount of space between an outline and the edge or border of an element.
